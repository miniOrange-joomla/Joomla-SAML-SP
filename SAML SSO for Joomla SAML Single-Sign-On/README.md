# Joomla SAML Single Sign-On (SSO) – Service Provider (SP) Plugin

Elevate your Joomla site's security and authentication with **Joomla SAML Single Sign-On (SSO)**. This powerful plugin allows your Joomla site to act as a **SAML 2.0 Service Provider (SP)**, enabling seamless login via Identity Providers (IDPs) such as **Azure AD**, **Okta**, **miniOrange**, **Google Workspace (GSuite)**, **Salesforce**, and many more.

## Key Highlights:
- **Multiple SAML Providers**:  
  Configure various IDPs like Azure AD, Google Apps, miniOrange, Keycloak, and others for flexible authentication.
- **Limitless Authentication**:  
  Supports unlimited SAML-based user logins.
- **Compatibility Across Joomla Versions**:  
  Works seamlessly with **Joomla 3, 4, and 5**.
- **Enhanced Security**:  
  Provides **encrypted and signed SAML responses** for secure authentication.

---

## Versions & Features:

### **Free Version**
- Limited user registration and authentication  
- Secure SSO authentication  
- Basic attribute mapping  

### **Standard Version**
- Unlimited user authentication  
- Basic group mapping  
- Custom redirect URLs  

### **Premium Version**
- Advanced attribute and group mapping  
- Backend login protection  
- **Single Logout (SLO)** support  

### **Enterprise Version**
- Multi-IDP support  
- Domain mapping  
- Custom certificates  
- Real-time IDP configuration auto-sync

## Use Cases:
- **Azure AD SSO**:  
  Enable Joomla login via Azure AD, Office 365, or Azure AD B2C.
- **Google Apps Integration**:  
  Allow users to log in using Google Workspace credentials.
- **AWS Cognito Integration**:  
  Enable seamless SSO with AWS Cognito and linked accounts.
- **LMS Platforms**:  
  Integrate SSO for learning platforms such as Moodle, Canvas LMS, and others.

---

## Add-Ons:
- **Page & Article Restriction**: Control access to specific Joomla pages or articles.
- **Role-Based Redirection**: Redirect users based on assigned Joomla roles/groups.
- **Media Restriction**: Block unauthorized access to files and folders.
- **SCIM User Provisioning**: Sync user changes from IDPs to Joomla in real time.

---

## Quick Setup:
1. Go to **Extensions → Manage → Install**.
2. Search for **"miniOrange SAML SP"** and install the plugin.
3. Configure via  
   **Component → miniOrange SAML SP → Service Provider Setup**.
