# ADFS SAML Single Sign-On (SSO) for Joomla

Simplify authentication on your Joomla site with **ADFS SAML Single Sign-On (SSO)**, providing secure login using **SAML 2.0** for your ADFS accounts.

## Key Highlights:
- **Secure & Seamless Login**:  
  Authenticate Joomla users with ADFS credentials. Users already logged into other ADFS-connected applications can access Joomla without re-entering credentials.
- **Windows SSO Supported**:  
  Integrated Windows Authentication for seamless, enterprise-grade access.
- **Multiple ADFS Servers**:  
  Connect your Joomla site to multiple ADFS servers for flexible authentication.

---

## Free Version Features:
- **IDP Credential Login**:  
  Secure login using SAML 2.0–compliant Identity Provider (IDP) credentials.
- **Automatic User Registration**:  
  Automatically registers users after successful login if they do not already exist in Joomla.
- **Simple Setup**:  
  Configure with any IDP using a metadata file or URL in a single click.
- **Basic Group Mapping**:  
  Assign default Joomla groups to users upon login.
- **Export Configuration**:  
  Download configuration files for easy upgrades or migration.

---

## Upgraded Versions:

### **Licensed Version Features**
- **Multiple SAML IDPs**:  
  Unlimited user authentication and creation with support for multiple IDPs and custom SP certificates.
- **Single Logout (SLO)**:  
  Log out from all connected applications with a single click.
- **Auto-Redirect to IDP**:  
  Restrict access to authenticated users and redirect unauthenticated users to the IDP.
- **Advanced Attribute & Group Mapping**:  
  Map IDP attributes (username, email, roles) to Joomla fields and groups.
- **Force Authentication**:  
  Redirect users to the IDP for re-authentication even if an active session exists.

## Get Started:
1. Go to **Extensions → Manage → Install**  
2. Search for **"ADFS SAML SSO"** and install the plugin  
3. Configure via **Component → ADFS SAML SSO → Plugin Setup**
