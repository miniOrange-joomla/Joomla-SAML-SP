# Joomla G Suite SAML Single Sign-On (SSO) Plugin

Enhance your Joomla site's security with **Joomla SAML Single Sign-On (SSO)**. This plugin enables seamless login using **Google Apps (G Suite)** credentials or any other **SAML 2.0–compliant Identity Provider (IDP)**. It is ideal for enterprises, schools, and organizations looking to streamline authentication processes.

## Key Highlights:
- **Easy Integration**:  
  Supports Google Apps (G Suite) SSO with attribute mapping such as first name, last name, and employee details.
- **Automatic User Registration**:  
  Automatically registers new users after successful SSO login.
- **Broad Compatibility**:  
  Works with many SAML-compliant IDPs including **Azure AD**, **Keycloak**, **Okta**, **ADFS**, and more.

---

## Free Version Features:
- Login using IDP credentials  
- Automatic user registration for new users  
- Basic group mapping to assign default Joomla groups on login  
- Export configuration for easy upgrades  

---

## Upgraded Versions:

### **Licensed Version Features**
- **Support for Multiple SAML IDPs**:  
  Authenticate users from multiple IDPs simultaneously.
- **Unlimited Users**:  
  Unlimited user creation and authentication with multiple IDPs.
- **Single Logout (SLO)**:  
  Log out from all connected applications with a single click.
- **Auto-Redirect to IDP**:  
  Restrict protected pages to authenticated users only.
- **Backdoor Superuser Login**:  
  Secure fallback access for Joomla administrators if IDP access is unavailable.
- **Advanced Attribute & Group Mapping**:  
  Map IDP attributes (username, email, roles) to Joomla fields and groups.
- **Integrated Windows Authentication**:  
  Supports Integrated Windows Authentication (with ADFS).

## Get Started:
1. Go to **Extensions → Manage → Install**
2. Search for **"Joomla SAML Single Sign-On SSO"**
3. Configure under **Component → SSO Plugin → IDP Setup**
